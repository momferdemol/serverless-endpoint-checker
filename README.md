## Endpoint Checker

Just a playground project to tinker with AWS resources.

- API Gateway
- Lambda
- EventBridge

**Dependency**

Python dependencies are listed in `pyproject.toml` and installed with poetry.

**Development**

Local development environment created with `devenv.nix`.
